# MJ plumber and heating — Website

Generated by The Virtual Factory site exporter.

## Quick start

```bash
npm install
npm run dev
```

## Deploy

```bash
npm run build
npm run start
```

Or deploy to Vercel:

```bash
npx vercel
```

## Customisation

- **Colours**: Edit `src/data/lead.ts` — change `primaryColor` / `secondaryColor`.
- **Text/content**: Edit `src/data/lead.ts` — all site content is in the `lead` object.
- **Images**: Replace files in your R2 path `public-vfwebdesign/niches/plumber/`.
- **Template**: To switch template, change the import in `src/app/page.tsx`.
